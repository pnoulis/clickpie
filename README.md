## About

**clickpie** (clickup + ie) contains all information related to [intelligent entertainment]'s

## Getting started

### Prerequisites

- git

  version: 2.34.1

- make
  
  version: 4.3

### Installation

```sh
git clone git@github.com:pnoulis/clickpie.git
make
```
